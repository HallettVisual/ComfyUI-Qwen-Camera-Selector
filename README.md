# ComfyUI Qwen Camera Selector

Simple camera prompt builder node for **Qwen-Edit-2509 Multiple-Angles** in ComfyUI.

- Groups: core movement, orbit rotation, vertical view, lens framing, combined actions
- Toggle to **keep character consistency**
- Toggle to **enable light restoration**  
- Free text field for extra instructions

## Installation

1. Download this repository as a ZIP.
2. Extract the folder into your `ComfyUI/custom_nodes` directory.
3. Restart ComfyUI.
